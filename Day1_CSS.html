Introduction of CSS (CaseCading Style-Sheet)

What is CSS? CSS is a style-sheet language used to describe the look and presentation of our document which is written in HTML or XML. It defines how elements are displayed on a web-page
It makes our website visually and appealing and user-friendly.
Creators and Css Maintencnace? CSS was created by a group of individuals known as the (World Wide Web Consortiun)_---W3C
CSS History of CSS to CSS3 CSS1(1996)-Limited properties and selectors CSS2(1998)-It was not released officially CSS3(2011)-It is the advance of version of css
CSS rule and structure 1) A CSS consist of a selector and a declearation block
2) Declaration are used to define the style properties for the selected elements
3) Each declaration consist of a property and a value 
    selector{
        property: value;
        /*More property and values*/
    }
    Example-:(Syntax of CSS3)
    h1{
        color: red;
        font-size: 32px;
    }

What are selectors in CSS Generally selectors are used to select that element to be style
They are mainly three types 
            1) Tag/Element selector
            Eg-: 
            h1{
                color: red;
                font-size: 32px;
            }
            2) Class selector- it is emplimented using (.btn)
            Eg-: 
            .bnt{ //btn is a class here//
                color: red;
                font-size: 32px;
            }
            3) Id selector -: It is implemented by using (#) symbol
            Eg-:
            #bnt{ //btn is a ID  here bcz of # symbol//
                color: red;
                font-size: 32px;
            }
            4) Pseudo selector -: It is implemented by using (:) symbol
            Eg-:
            #bnt:hover{ //btn is a ID  here bcz of # symbol//
                color: red;
                font-size: 32px;
            }
